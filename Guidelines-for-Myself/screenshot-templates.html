Word Pop Screenshot Templates

This page contains templates for all the required screenshots for your Word Pop game. Each template below represents one of the screenshots you need to take. Take a screenshot of each frame to create the assets for your portfolio.

For the best results:

  • Use Cmd + Shift + 4, then Space to capture the specific frame
  • Save the file with the correct name as indicated (welcome.png, gameplay.png, etc.)
  • Place all screenshots in the /screenshots directory
